# Lógica de Programación 1

Programa en JavaScript que solicita 3 números al usuario, valida que sean numéricos, los ordena de mayor a menor y de menor a mayor, e identifica si los números son iguales.

## Archivos
- `index.html`
- `app.js`

## Funcionalidades
- Solicita 3 números por `prompt`
- Valida que sean números
- Ordena de mayor a menor
- Ordena de menor a mayor
- Detecta si los 3 números son iguales

## Pruebas
Ejemplos probados:
- 4, 4, 2
- 4, 2, 4
- 2, 4, 4
- 5, 5, 5
- 9, 1, 5
- -3, -1, -7
